Expand description

Connection for pre play packets Gets dropped by incoming_packet_task if closed or upgradet to play connection

Fields§

§id: u64

The unique ID of the client.

§gameprofile: Mutex<Option<GameProfile>>

The client’s game profile information.

§protocol: Arc<AtomicCell<ConnectionProtocol>>

The current connection state of the client (e.g., Handshaking, Status, Play).

§address: SocketAddr

The client’s IP address.

§cancel_token: CancellationToken

A token to cancel the client’s operations. Called when the connection is closed. Or client is removed.

§outgoing_queue: UnboundedSender<EnqueuedPacket>

A queue of serialized packets to send to the network

§network_writer: Arc<Mutex<TCPNetworkEncoder<BufWriter<OwnedWriteHalf>>>>

The packet encoder for outgoing packets.

§server: Arc<Server>

The shared server state.

§challenge: AtomicCell<[u8; 4]>

The challenge sent to the client during login.
